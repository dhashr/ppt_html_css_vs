What's the Difference?

HTML

Hpertext Markup Language

  Create the structure

  • Controls the layout of the content
  • Provide the structure for web page design
  • The fundamental building block of any web page

CSS

Cascading Style Sheet

  Stylze the website

  • Applies style to the web page element
  • Target various screen size to make web page responsive
  • Primarly handles the "Look and Feels"of a web page

Javascript

  Increase Interactivity

  • Adds interactivity to a web page
  • Handles complex functions and features
  • Programmatic code which enhances functionality
